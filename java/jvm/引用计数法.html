    # 引用计数算法

    很多教科书判断对象是否存活的算法是这样的，给对象中添加一个引用计数器，每当有一个地方引用他时，计数器的只就+1；每当引用失效时，计数器的致就-1；任何时刻计数器为 0 的对象就是不可能再被使用的。

    客观的说，引用计数算法的实现简单，判定的效率也很高，在大部分的情况下都是一个不错的算法，但是主流的 Java 虚拟机中没有选用应用计数法来管理内存，起主要原因是他很难解决对象之间相互引用的问题。
